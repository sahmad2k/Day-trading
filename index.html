Machine Learning Project — Predicting Short Opportunities

Curious about how data science can complement education? Explore the accompanying Python project that downloads real market data, engineers momentum/volatility features, and trains a Random Forest classifier to flag potential short setups (educational use only!).

  1. Install dependencies: pip install -r requirements.txt or run pip install yfinance pandas numpy scikit-learn matplotlib tqdm.
  2. Run the pipeline with defaults:
    python predict_short_opportunities.py
  3. Customize tickers/dates:
    python predict_short_opportunities.py --tickers AAPL MSFT NVDA --start 2018-01-01 --end 2024-01-01

The script prints cross-validation metrics and saves a feature importance chart to outputs/feature_importance.png so you can visualize which indicators mattered most.
